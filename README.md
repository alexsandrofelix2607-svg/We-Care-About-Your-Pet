# Nós Nos Importamos - Com o Seu Pet 🐾

Landing page responsiva para clínica veterinária e pet shop. 
Projeto focado em design moderno, boa experiência do usuário e conversão.

![Preview do Projeto](https://github.com/alexsandrofelix2607-svg/We-Care-About-Your-Pet/blob/main/img/pet.PNG?raw=true)

## **📋 Sobre o Projeto**
O "We Care" é uma landing page desenvolvida para uma empresa de cuidados com pets. 
O objetivo é apresentar os serviços de forma clara, profissional e transmitir 
confiança para tutores que buscam o melhor para seus animais.

Este projeto faz parte dos meus estudos de HTML5 e CSS3.
